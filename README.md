# 7 a 0 · Edición River ⚪🔴
Un juego de Juniloyd.

Tirá el dado, armá tu River histórico ideal con cracks de todas las épocas
(La Máquina, La Maquinita, el bicampeón del 75, el tri del 80, el 86, el 96 de
Francescoli, Los 4 Fantásticos, la era Gallardo...), elegí un DT campeón de River
y ganá la Copa de la Liga enfrentando a los 46 equipos más históricos del fútbol
argentino.

## CÓMO PUBLICARLO (GitHub + Vercel)
1. Subí el index.html a la RAÍZ de un repo nuevo en GitHub.
2. En Vercel: Add New -> Project -> importás el repo -> Deploy.
3. Listo, te da la URL para compartir.
